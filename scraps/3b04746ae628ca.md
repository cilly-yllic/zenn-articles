# Cloud StorageのAutoClassを設定していると、ライフサイクルの削除条件にストレージクラスを指定できなくなった。

`Error: googleapi: Error 400: Buckets cannot have both Autoclass and Lifecycle DELETE rules with matches_storage_class condition., invalid`

```
resource "google_storage_bucket" "default" {
  name          = var.name
  location      = var.region
  provider      = google-beta

  autoclass {
    enabled = true
  }

  lifecycle_rule {
    action {
      type = "Delete"
    }
    condition {
      age                   = 7 * 365
      matches_storage_class = ["ARCHIVE"]
    }
  }
}
```

多分これならいけるということだと思う（まだ動作確認していない）

```
resource "google_storage_bucket" "default" {
  name          = var.name
  location      = var.region
  provider      = google-beta

  autoclass {
    enabled = true
  }

  lifecycle_rule {
    action {
      type = "Delete"
    }
    condition {
      age                   = 7 * 365
    }
  }
}
```
