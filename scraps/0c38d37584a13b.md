# 【メモ】TypeScript: ESMを実行する
## 2024-02-07

```
"scripts": {
  "run": "ts-node-esm ./index.ts"
}
```

先日まではこれで動いていたが、  
`TypeError [ERR_UNKNOWN_FILE_EXTENSION]: Unknown file extension`  
でうごかなくなった。

```
"scripts": {
  "1": "node --loader ts-node/esm ./index.ts",
  "2": "tsx ./index.ts",
  "3": "npx tsx ./index.ts"
}
```

`node --loader ts-node/esm`か `npm i -D tsx`で `tsx`を使うかでサクッと対応できる。
