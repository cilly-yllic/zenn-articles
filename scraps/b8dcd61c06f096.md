# 【メモ】TypeScriptの便利な型あれこれ
## 2023-12-23

Union型をインターセクション型に変換

```
type UnionToIntersection<Union> = (Union extends any ? (k: Union) => void : never) extends (k: infer Intersection) => void
  ? Intersection
  : never;
```

```
type Hoge = string | number
const hoge: UnionToIntersection<Hoge> = '' // TS2322: Type string is not assignable to type never

type Foo = { aa: string } | { bb: string }
const foo: UnionToIntersection<Foo> = { aa: '', bb: '' } //  { aa: string; } & { bb: string; }
```

などで拡張子を指定する時に便利かも（かなり限定的だけど💦）

```
type UnionToIntersection<Union> = (Union extends any ? (k: Union) => void : never) extends (k: infer Intersection) => void
  ? Intersection
  : never;

type UnionToFunctionalIntersections<Union> = UnionToIntersection<Union extends any ? (u: Union) => void : never>;

type PopUnion<Union> = UnionToFunctionalIntersections<Union> extends (a: infer A) => void ? A : never;

export type UnionJoinWithSeparator<Union extends string, Separator extends string> = PopUnion<Union> extends infer Last
  ? Last extends string
    ? Exclude<Union, Last> extends never
      ? Last
      :
      | `${UnionJoinWithSeparator<Exclude<Union, Last>, Separator>}${Separator}${Last}`
      | UnionJoinWithSeparator<Exclude<Union, Last>, Separator>
      | Last
    : never
  : never;
```

```
export const EXTENSIONS = {
  gif: '.gif',
  jpg: '.jpg',
  jpeg: '.jpeg',
  png: '.png',
  svg: '.svg',
} as const

type Accept = typeof EXTENSIONS[keyof typeof EXTENSIONS]
// ".gif" | ".jpg" | ".jpeg" | ".png" | ".svg" | ".gif,.jpg" | ".gif,.jpeg" | ".jpg,.jpeg" | ".gif,.jpg,.jpeg" | ".gif,.png" | ".jpg,.png" | ...
const accept: UnionJoinWithSeparator<Accept, ','> = `.gif,.jpeg`
```

*Union型の順番に繋げる必要がある（今回の場合 `'.jpg,.gif'`だとエラーになる）*
